# Gasoline Demand in Brazil: A State-Level Panel Approach

<p align="right"><a href="#portuguese">🇧🇷 Ler em Português</a></p>

Replication repository for the doctoral thesis:

> **Quantifying the impacts of electrification and taxation on gasoline demand in Brazil: a state-level panel modeling approach with classification of heterogeneous demand regimes**
> Sergio Mauricio Prolo Santos Junior — PPGEAS/UFSC, 2026

A companion paper covering the core results through 2021 has been peer-reviewed and published:

> Prolo, S., Pellegrini, A., Kraus Jr., W., Coelho, A. H., & Rose, J. M. (2025). Estimating gasoline demand and electric vehicle adoption: A spatial model for Brazil. *Research in Transportation Economics*, 113, 101623. https://doi.org/10.1016/j.retrec.2024.101623

---

## Overview

This repository contains all code, raw data, and outputs required to reproduce the empirical results of the thesis. The analysis estimates short- and long-run gasoline demand elasticities across all 27 Brazilian federal units using an 18-year balanced panel (2006–2023) and projects future consumption through 2060 under alternative fuel taxation and vehicle electrification scenarios.

The key methodological contributions are:

- **Iterative FGLS estimation** with panel-specific heteroskedastic error variances and AR(1) autocorrelation, implemented in Stata via `xtgls`.
- **Spatial spillover of EV adoption** captured through a binary queen-contiguity weight matrix.
- **K-means classification** of Brazilian states into three demand regimes based on the estimated error-structure parameters.

## Workflow

The full pipeline runs in three stages.

### 1 — Build the panel dataset (Python)

```bash
python dataset.py
```

This script assembles the complete panel from the raw input files. It reads fuel sales from ANP, deflates prices using IPCA, constructs ICEV and EV motorization indexes (using the Gompertz scrappage model for EV stock prior to 2014), computes the spatial weight matrix, and saves `data/output/data_2023.csv` and `stata_scripts/data_stata_pre.xlsx`.

If you need to rebuild the circulating fleet estimates independently:

```bash
python frota_circulante.py
```

### 2 — Estimate the model (Stata 18)

Run the scripts in order from the `stata_scripts/` directory:

```stata
do import.do        // Load data, declare panel, create dummies
do tests.do         // Unit root and cointegration tests
do regress.do       // FGLS estimation; exports fgls_results.xlsx
```

The `regress.do` script estimates the preferred model (M0) using `xtgls` with `panels(het)` and `corr(psa1)`, and exports four sheets to `fgls_results.xlsx`: predictions, panel error parameters (ρ̂ and σ̂), coefficient vector β̂, and the full covariance matrix.

### 3 — Generate figures (Python)

```bash
python k_means.py          # Cluster analysis of (σ̂, ρ̂) — Figure 8
python scenario_compare.py # Scenario bar chart and confidence intervals — Figures 15–16
python cloropeth.py        # Choropleth maps of per-capita sales — Figure 1
```

## Visão Geral

Este repositório contém todo o código, dados brutos e outputs necessários para reproduzir os resultados empíricos da tese. A análise estima elasticidades de demanda por gasolina de curto e longo prazo para as 27 unidades federativas brasileiras a partir de um painel equilibrado de 18 anos (2006–2023) e projeta o consumo futuro até 2060 sob cenários alternativos de tributação de combustíveis e eletrificação veicular.

As principais contribuições metodológicas são:

- **Estimação iterativa por FGLS** com variâncias heterocedásticas e autocorrelação AR(1) específicas por painel, implementada no Stata via `xtgls`.
- **Efeito de transbordamento espacial da adoção de VEs** capturado por uma matriz de pesos de contiguidade do tipo rainha.
- **Classificação k-means** dos estados brasileiros em três regimes de demanda com base nos parâmetros estimados da estrutura de erros.

## Fluxo de Trabalho

O pipeline completo é executado em três etapas.

### 1 — Construção do painel (Python)

```bash
python dataset.py
```

Este script monta o painel completo a partir dos arquivos brutos. Ele lê as vendas de combustíveis da ANP, deflaciona os preços pelo IPCA, constrói os índices de motorização de ICEV e VE (usando o modelo de sucateamento Gompertz para o estoque de VEs anterior a 2014), calcula a matriz de pesos espaciais e salva `data/output/data_2023.csv` e `stata_scripts/data_stata_pre.xlsx`.

Para reconstruir somente as estimativas da frota circulante:

```bash
python frota_circulante.py
```

### 2 — Estimação do modelo (Stata 18)

Execute os scripts na ordem indicada, a partir do diretório `stata_scripts/`:

```stata
do import.do        // Carrega os dados, declara o painel e cria as dummies
do tests.do         // Testes de raiz unitária e cointegração
do regress.do       // Estimação FGLS; exporta fgls_results.xlsx
```

O script `regress.do` estima o modelo preferido (M0) com `xtgls panels(het) corr(psa1)` e exporta quatro abas para `fgls_results.xlsx`: previsões, parâmetros de erro por painel (ρ̂ e σ̂), vetor de coeficientes β̂ e a matriz de covariância completa.

### 3 — Geração das figuras (Python)

```bash
python k_means.py          # Análise de agrupamento de (σ̂, ρ̂) — Figura 8
python scenario_compare.py # Gráfico de barras dos cenários e intervalos de confiança — Figuras 15–16
python cloropeth.py        # Mapas coropléticos de vendas per capita — Figura 1
```
